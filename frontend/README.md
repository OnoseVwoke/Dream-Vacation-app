# Dream Vacation App

## Requirements

- Docker Desktop
- Git
- VS Code

## Setup

Clone the repository:

```bash
git clone <repo-url>
```

Move into project:

```bash
cd Dream-Vacation-app
```

## Start Application

```bash
docker-compose up --build
```

## Stop Application

```bash
docker-compose down
```

## Services

Frontend:

http://localhost:3000

Backend:

http://localhost:5000

Database:

PostgreSQL
